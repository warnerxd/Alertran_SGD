 ALERTRAN_SGD. - Sistema de Gestión de Desviaciones, una aplicación de escritorio desarrollada en Python que automatiza la creación masiva de desviaciones en el sistema Alertan.

Requisitos 


Paso 1
tener instalacion de  
python --version  # Python 3.8 o superior

Paso 2 
Link Repo clonar 
git clone https://github.com/TU-USUARIO/Alertran_SGD.git

Paso 3
cd Alertran_SGD

Paso 4
Creacion nuevo estorno virtual
python -m venv venv

Paso 5
Activacion entorno virtual
venv\Scripts\activate

Paso 6
las dependecias ya estan incluidas en el archivo requirements.txt
pip install -r requirements.txt


Paso 7
python main.py
